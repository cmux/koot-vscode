-   右键点击任意目录时，可快捷创建 React 组件

## 使用 _Koot.js_

_Koot.js_ 是一个基于 React 的开发框架，支持 SSR 项目和 SPA 项目的开发。

**亮点:**

-   零配置支持热更新
-   零配置支持 _TypeScript_ 开发
-   零配置支持离线缓存
-   CSS 命名空间
-   高级多语言支持
-   路由按需载入

开始使用: [Koot.js 上手指南 & 官方文档](https://koot.js.org/)
